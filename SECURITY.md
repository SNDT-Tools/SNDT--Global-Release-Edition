# Security Policy

## Supported Versions

Currently, only the latest release of SNDT is actively supported with security updates and patches.

| Version | Supported          |
| ------- | ------------------ |
| 0.3.x   | :white_check_mark: |
| < 0.3.0 | :x:                |

## Reporting a Vulnerability

Security is a top priority for network diagnostics tools. If you discover a security vulnerability within SNDT, please do not disclose it publicly in the Issues tab. 

Please report any security-related issues directly by contacting the repository owner or opening a private security advisory. All legitimate vulnerabilities will be investigated and addressed as quickly as possible.
